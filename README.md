# Técnicas Computacionais 
Repositório da disciplina "INT0046 - Técnicas Computacionais" da UFPE. Serão disponibilizados os códigos utilizados nas atividades realizadas pelos estudantes, a fim de que tenhamos uma experiência baseada em ferramentas e processos utilizados rotineiramente.

## Materiais recomendados
1. [Introdução ao GitHub](https://lab.github.com/githubtraining/introduction-to-github)
2. [Stack Overflow](https://stackoverflow.com/)
3. [Google Colab](https://colab.research.google.com/)
4. [Automating the Boring Stuff with Python - Al Sweigart](https://automatetheboringstuff.com/)
5. [Think Python - Allen Downey](https://greenteapress.com/thinkpython2/thinkpython2.pdf)
6. [Data Structures and Information Retrieval in Python - Allen Downey](https://greenteapress.com/wp/data-structures-and-information-retrieval-in-python/)
7. [Codility - Developer Training](https://app.codility.com/programmers/)
8. [Stack Overflow 2021 Developer Survey](https://insights.stackoverflow.com/survey/2021)
